# Okuma React

Bu proje Vite + React (JavaScript) ile oluşturulmuş basit bir iskelet.

PowerShell üzerinde çalıştırmak için:

```powershell
npm install
npm run dev
```

- `npm run dev`: Geliştirme sunucusunu başlatır
- `npm run build`: Üretim için paketler
- `npm run preview`: Üretim paketini önizler

Sonraki adım: Lütfen dönüştürmemi istediğiniz orijinal dosyayı buraya yapıştırın veya `src/original-file` yolunda bırakın; ben de onu ayrı bileşen dosyalarına böleceğim.
